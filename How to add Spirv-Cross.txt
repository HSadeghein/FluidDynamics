1.Download the SPIRV-Cross project by .gitmodule
	a. Open your git bash in the soloution directory.
	b. Wrirte "git submodule init" command.
	c. Write "git submodule update" command.
2.Run the checkout_glslang_spirv_tools.sh bash file.
3.Run the build_glslang_spirv_tools.sh bash file.
4.You can access to the glslangValidator.exe in {$SoloutionDir}/vendors/SPIRV-Cross/external/glslang-build/output/bin.
5.Build the SPIRV-Cross with cmake.
-----> mkdir build
-----> cmake ..
-----> cmake --build .

6.You can access to the spirv-cross.exe in {$SoloutionDir}/vendors/SPIRV-Cross/build/Debug.
7. At last shaders type must be set in VS ---> properties
